Machine learning tasks

  • Classification: Predict a class
  • Regression: Predict a continuous value
  • Clustering: Predict similarity

Machine learning setups

  • Supervised: Labelled data
  • Unsupervised: No labels
  • Semi-supervised: Small amounts of labelled data

Popular algorithms

  • Support vector machines (SVM)
  • Random Forests
  • Penalized Regression
  • K-means, K-nearest-neighbour
  • Deep neural networks

Data preprocessing

  • Some ML algorithms are sensitive to scales, so scale/center
  • Reduce dimensionality (PCA)

Training and Testing

  • Input data is split 70:30. First set is used to build model, second to evaluate overfitting.
  • K-fold cross validation repeats this K times
  • Leave one out (LOO) or leave P out (LPO)
